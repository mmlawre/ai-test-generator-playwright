# Sample User Stories (Demo Web App)

## Login
As a user, I want to log in so I can access the dashboard.

Acceptance criteria:
- If username/password is valid, user lands on /dashboard and sees "Welcome"
- If invalid, user sees error banner "Invalid credentials"

## Settings
As a logged-in user, I want to update my profile settings.

Acceptance criteria:
- User can update display name
- Success message appears "Saved"
- If display name is empty, show validation message "Required"
